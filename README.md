## My Note

- Format all the markdown files.

  - Add settings in `.prettierrc`.

    ```json
    {
      "printWidth": 80,
      "proseWrap": "always"
    }
    ```

  - Excute the command.

    ```bash
    npx prettier --write "**/*.md"
    ```
